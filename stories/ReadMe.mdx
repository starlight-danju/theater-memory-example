> 문서 작성 예시입니다. Nextjs는 리액트 기반의 프로젝트입니다. 따라서 컴포넌트 주도 개발을 하게 됩니다. 컴포넌트 주도 개발 프로젝트를 문서화 시키기 적합한 스토리북을 사용하겠습니다.

> 예시에는 nextjs 프로젝트에서 만든 컴포넌트에서 가져오기 쉬운 두 가지만 간단히 표현했습니다. 사실 스토리북을 사용할 생각이라면 독립적인 컴포넌트를 먼저 작성하시기 바랍니다. 다른 모듈을 Import 하는 부분이 stories 폴더
> 안에서 경로가 지정되어야 합니다.
